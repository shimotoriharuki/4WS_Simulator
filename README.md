# ファイルの説明

## parameters.m
4WSロボットのパラメータを定義したファイルです．このファイルを実行してからシミュレーションを実行する必要があります．

## simulator.slx
4WSのシミュレーションファイルです．ロボットの速度とステアの角度を指定してシミュレーションすることができます．

## zero_gravity_simulation.slx
4WSロボットを無重力みたいに動かそうとしているファイルです．外力からロボットの加速度と進行方向を計算して動かします．
